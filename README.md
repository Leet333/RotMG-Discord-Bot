# RotMG Discord Bot
Discord bot with RotMG related commands

## Requirements
```
discord.py
python3
aiohttp
aiosqlite
asyncio
```

## Features
* Player lookup
* Guild lookup
* Item lookup
* Character viewer
* Random class selector
* O3 Headcount
